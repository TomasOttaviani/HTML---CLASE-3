Etiquetas de formato

Las etiquetas les dan estructura a nuestro archivo.

Entre ellas vamos a encontrar las siguientes etiquetas:

  • Etiqueta "P"

    • Nos genera un bloque de texto/parrafo que va de principio a fin de la pantalla.

  • Etiquetas "H1"

    • Viene de header que es el encabezado/titulo de nuestro archivo. Esta etiqueta tiene van de "H1" a "H6" y definen el impacto del titulo, siendo "H1" el mas importante y "H6" el menos importante visualmente.

  • Etiqueta "OL"

    • Nos genera una lista ordenada (Enumerandolas), dentro de ella se encuentran los items que completan la lista que va acompañado de la etiqueta "LI".

  • Etiqueta "UL"

    • Nos genera una lista que no lleva un orden, es decir que no necesariamente quien esta primero en la lista es el primer paso a seguir, no tienen un orden establecido. Dentro de ella tambien hay que ingresar los items con la etiqueta "LI"

  • Etiqueta "HR"

    • Esta etiqueta sirve para generar una barra que ocupa un bloque entero dentro de la pantalla y puede usarse para separar secciones de una noticia. La particularidad de esta etiqueta es que se cierra a ella misma.

  • Etiqueta "SPAN"

    • Es una etiqueta de texto, pero a diferencia de la etiqueta de "P" esta no ocupa la totalidad de la pantalla, si no que seguido de otra etiqueta, esta se posicionaria alado de la misma y no por debajo

Etiquetas de bloques

Estas etiquetas sirven para darle un orden dentro al archivo dentro del codigo. No tienen un cambio de impacto visual.

Entre ellas vamos a encontrar las siguientes etiquetas:

  • Etiqueta "HEADER"

    • Dentro de esta etiqueta se va a encontrar todo el encabezado de la pagina, como por ejemplo el titulo.

  • Etiqueta "MAIN"

    • Aca vamos a encontrar todo el contenido principal.

  • Etiqueta "FOOTER"

    • Se encuentra siempre por debajo de todo el contenido principal. Es un apartado donde normalmente se puede encontrar informacion adicional link a redes sociales o telefonos de contacto.

  • Etiqueta "SECTION"

    • Separa las secciones de una pagina, por ejemplo la seccion de HTML con la seccion de otro lenguaje como CSS

  • Etiqueta "ARTICLE"

    • Separa los articulos de la seccion, por ejemplo las etiquetas de formato y las etiquetas de bloques

  • Etiqueta "DIV"

    • etiqueta universal que sirve para agrupar elementos, por ejemplo 2 textos que mas adelante voy a querer editar o resaltar y poder adjuntarlos ya que estan juntos por la etiqueta div.

Etiquetas de contenido

Estas etiquetas sirven para marcar contenido

Entre ellas vamos a encontrar las siguientes etiquetas:

  • Etiqueta "IMG"

    • Con esta etiqueta podes agregar imagenes a tu web. Para eso hay que tener en cuenta que lleva un atributo "SCR"

  • Etiqueta "A"

    • Genera un Link, que se accede dandole click a lo destacado. Lleva un atributo "HREF"

  • Etiqueta "FORM"

    • Dentro de ella se encuentran las etiquetas propias de un formulario. Por Ejemplo: Etiqueta "INPUT" y Etiqueta LABEL"

  • Etiqueta "INPUT"

    • Sirve para que el usuario pueda interactuar con la pagina, ingresando texto, seleccionando opciones o demas. Siempre va acompañada del atributo "TYPE" que sirve para que la computadora limite al usuario a saber que tiene que ingresar.

  • Etiqueta "LABEL"

    • Esta sirve para informar al usuario que tiene que ingresar. Tambien se acompaña de un atributo "FOR" y este almacena la informacion ingresada por el usuario

  • Etiqueta "BOTTON"

    • Genera un boton para que el usuario puede clickear
